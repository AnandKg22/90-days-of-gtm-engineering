# GTM Open-Source GitHub Repositories

Open-source repositories providing core components for database replication, workflow automation, and AI agent frameworks.

## 💻 Key Repositories

*   **n8n-io/n8n**: Fair-code workflow automation node server. Excellent starter repository for hosting local ingestion pipelines.
*   **dbt-labs/dbt-core**: Analytics engineering transformations engine. Standard tool for SQL pipeline modeling and data testing.
*   **supabase/supabase**: Open-source Postgres platform providing backend authentication, pgvector vector databases, and REST APIs.
*   **pgvector/pgvector**: Open-source vector similarity search extension for PostgreSQL database instances.
*   **langchain-ai/langgraph**: Orchestration framework for building stateful, multi-agent AI systems with clean memory loops.
