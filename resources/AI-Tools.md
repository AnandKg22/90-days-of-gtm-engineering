# AI Tools for GTM Systems

The core generative AI APIs, tracing utilities, data enrichers, and Model Context Protocol integrations.

## 🤖 AI Stack Tools

*   **OpenAI API & Anthropic API**: Core foundational LLM engines for personalizing outbound SDR emails and parsing customer tickets.
*   **LangSmith & Phoenix**: Debugging and observability dashboards to trace agent node state transitions and audit latency.
*   **Clay.run**: B2B data enrichment aggregator programmatically querying ZoomInfo, Apollo, and LinkedIn APIs to build clean leads datasets.
*   **Model Context Protocol (MCP)**: Open-source specification connecting local desktop developer IDEs to LLMs and database executors.
*   **LlamaIndex**: Data ingestion utilities to build clean semantic indices for RAG Knowledge Assistants.
