# Contributing to 90 Days of GTM Engineering

We welcome contributions to this open-source curriculum! Whether you're fixing typos, expanding on explanations, sharing prompt templates, or offering workflow designs, feel free to open a Pull Request.

## How to Contribute
1. Fork the repository.
2. Create a new branch: `git checkout -b feature/your-improvement`.
3. Commit your changes: `git commit -am 'Add improvements to Day X'`.
4. Push to the branch: `git push origin feature/your-improvement`.
5. Submit a Pull Request.
