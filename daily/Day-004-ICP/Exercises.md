# Exercises - Day 004: Ideal Customer Profile (ICP) Designs

This document contains the completed exercises designing Ideal Customer Profiles for VivaExams, IMSGOA, and Marine Colleges.

---

## 1. VivaExams B2B Enterprise ICP

| ICP Metric | Target Profile |
| :--- | :--- |
| **Industry** | Maritime Education Institutes, Professional Exam Prep Centers, Shipping Academies |
| **Company Size** | 20 – 150 employees (faculty + admin) |
| **Target Audience** | 500 – 5,000 active ship cadets per year |
| **Technographics** | Uses legacy Learning Management Systems (e.g., Moodle, Blackboard), Stripe (for student fees) |
| **Pain Points** | Manual question paper creation, tedious auto-grading, and cadet cheating in online mock tests |
| **Buying Signals** | Launching new certification courses, hiring exam supervisors, or opening secondary campuses |

---

## 2. IMSGOA Recruitment Sponsorship ICP

| ICP Metric | Target Profile |
| :--- | :--- |
| **Industry** | Commercial Ship Management Companies, Crewing Agencies, Ship Owners |
| **Company Size** | Global shipping companies managing 15+ commercial vessels (tankers, bulk carriers) |
| **Target Geography** | Singapore, Mumbai, Dubai, Greece, London |
| **Technographics** | Uses modern crewing software (e.g. SeaManager, ShipNet), Slack / MS Teams |
| **Pain Points** | Shortage of highly trained junior marine engineers, high onboard failure rates, and safety incidents |
| **Buying Signals** | Acquiring new cargo vessels, expanding dry-dock operations, or hiring marine training directors |

---

## 3. Marine Colleges DGS Compliance ICP

| ICP Metric | Target Profile |
| :--- | :--- |
| **Industry** | Post-Graduate Maritime Academies, Pre-Sea Training Colleges |
| **Company Size** | 50 – 300 employees |
| **Target Geography** | Coastal Indian cities (Chennai, Mumbai, Kolkata, Goa, Gujarat) |
| **Technographics** | Relies on manual Excel grading sheets, DGS-approved medical templates |
| **Pain Points** | Meeting strict Director General of Shipping (DGS) compliance standards, keeping student placement logs up to date |
| **Buying Signals** | Undergoing audit reviews, renewing DGS accreditations, or investing in maritime simulators |
